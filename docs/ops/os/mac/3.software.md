## 开发工具

- 前端
  
  ```shell
   # 安装包管理工具
   npm install -g n 
   n stable ## 把当前系统的 Node 更新成最新的 “稳定版本”
   n lts ## 长期支持版
   n latest ## 最新版
   n 14.16.1 ## 指定安装版本
  ```

- IDE/编辑器
  
    IDEA全家桶
  
    VS code
  
    HBuildX
