## 配置

### 配置网络

### 配置ssh

``` shell
# 安装ssh服务
yum install openssh-server
# 启动ssh服务
service sshd start
```

​	配置xshell连接linux终端

### 更新内核

~~~shell
yum update kernel 
yum update && yum upgrade
~~~

### 安装其他工具

~~~ shell
yum install wget
yum install p7zip 
yum install gcc
yum install java
yum install lrzsz

~~~

### 创建用户
useradd username  创建用户
passwd username  给已创建的用户设置密码
userdel username  删除用户


### 设置共享文件夹

~~~ shell
# 检查是否安装了samba
rpm -qi samba

# 安装samba
yum install samba

~~~
