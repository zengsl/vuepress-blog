### 2、配置ssh

``` shell
# 安装ssh服务
yum install openssh-server
# 启动ssh服务
service sshd start
```

​	配置xshell连接linux终端

### 3、更新内核

~~~shell
yum update kernel 
yum update && yum upgrade
~~~

### 4、安装其他工具

~~~ shell
yum install wget
yum install p7zip 
yum install gcc
yum install java
yum install lrzsz

~~~
